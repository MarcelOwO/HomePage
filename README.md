# Homepage

A Dotnet blazor application built using the Blazor WebAssembly Standalone App template.

This projects stands as my personal cv website.

It is currently hosted under the link [www.marcel-sigl.dev](https://www.marcel-sigl.dev) on azure.

## Building

To run the project for simple debugging run the following command.

``` bash

dotnet run

```


## Notes

As the project is made to only run on azures free tier it does not rely on any external services or backends. And also is not made to run on something like docker. 
However there are plans to migrate hosting from azure to a self hosted server.
